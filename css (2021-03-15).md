*CSS* : Cascading Style Sheet

*HTML* : Structure du document

*CSS* : Style, mise en forme

---
-> # Syntaxe <-

    h1 {
        color: blue;
        font-size: 24px;
    }
<br>

*sélecteur* : h1
<br>

*propriété* : color
<br>

*blue* : valeur
<br>

Références : [Moz://a Developer Network](https://developer.mozilla.org/fr/docs/Web/CSS)

---
-> # Où écrire le CSS ? <-

*dans un attribut style du HTML*

    <h1 style="color:blue;">Mon titre</h1>
<br>

*dans l'en-tête de la page HTML*

    <head>
        <style>
        h1 {
            color:blue;
        }
        </style>
    </head>
<br>

*dans un fichier séparé*

    <head>
        <link href="fichier.css" rel="stylesheet">
    </head>

---
-> # Les sélecteurs <-

*les éléments HTML*

    p {}
<br>

*les classes*

    <p class="important">...</p>
    .important {}
<br>

*les id*

    <p id="name">...</p>
    #name {}
<br>

*tous les éléments*

```
* {}
```
<br>

*les attributs*

    a[title] {}    
    a[href="http://www.papertoilet.com/"] {}

Les propriétés sont héritées des parents (<body> eg)
La priorité est donnée au sélecteur le plus spécifique

---
-> # Combiner les sélecteurs <-

*grouper les sélecteurs*

    h1, h2 {}
<br>

*éléments descendants*

    .info a {}
    /* s'applique aux sélecteurs a descendant de la classe info */
<br>

*éléments fils*

    .info > a {}
    /* s'apllique aux sécteurs a descendant directement de la classe info */

---
-> # Les pseudo-classes <-

Permet de cibler des éléments en fonction de leur état

    a:visited {}
    div:hover {}
    li:nth-child(odd) {}

---
-> # Les pseudo-éléments <-

Permet de cibler certaines parties de l'élément

    p::first-line {}
    p::first-letter {}
    p::selection {}
